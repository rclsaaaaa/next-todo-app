# TODO Management App

## Overview

This is a TODO management application that allows users to create, update, and delete tasks.
User authentication is implemented using NextAuth.js, and data is stored in PostgreSQL.

## App URL

- todo app：https://next-todo-app1974.vercel.app/

## Features

- User authentication (Sign up / Login / Logout)
- Create, update, delete TODOs
- User-specific task management
